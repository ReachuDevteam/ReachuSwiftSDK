# ReachuSwiftSDK 3.0.0

This release aligns documentation and example projects to the 3.x line. There are no functional changes over 2.0.0.

Highlights
- Docs: Update installation snippet to `from: "3.0.0"` in README
- Demos: Pin demo repos/projects to `ReachuSwiftSDK == 3.0.0`
- API: No changes from 2.0.0

Upgrade Notes
- If you are on 2.0.0, you can upgrade to 3.0.0 with no code changes.
- Demos should update their SPM pin to exactly `3.0.0` or a compatible range.
