## üé® Estructura del Archivo JSON

El archivo `reachu-config.json` define:

### **1. API Configuration**
```json
{
  "api": {
    "baseURL": "https://api.reachu.io",
    "apiKey": "your-api-key",
    "environment": "development"
  }
}
```

### **2. Theme (Light/Dark)**
```json
{
  "theme": {
    "name": "TV2 Dark Theme",
    "mode": "dark",
    "darkColors": {
      "primary": "#7B5FFF",
      "background": "#1A1625",
      "surface": "#2B2438"
    }
  }
}
```

### **3. Typography**
```json
{
  "typography": {
    "fontFamily": "System",
    "sizes": {
      "largeTitle": 32,
      "title": 24,
      "body": 16
    }
  }
}
```

### **4. Spacing & Border Radius**
```json
{
  "spacing": {
    "xs": 4,
    "sm": 8,
    "md": 16
  },
  "borderRadius": {
    "small": 8,
    "medium": 12
  }
}
```

### **5. Features (Enable/Disable)**
```json
{
  "features": {
    "enableLiveStreaming": true,
    "enableProductCatalog": true,
    "enableCheckout": true
  }
}
```
